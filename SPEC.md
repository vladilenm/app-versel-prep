# Личный трекер бюджета

## Идея
Простое SPA-приложение для отслеживания личных расходов и доходов с разбивкой по категориям и визуализацией через графики.

## Стек
- **React 19** + **TypeScript** — UI
- **Vite** — сборка
- **Tailwind CSS 4** — стилизация
- **Recharts** — графики (pie chart, bar chart)
- **localStorage** — хранение данных (без бэкенда)
- **uuid** — генерация ID транзакций

## Функциональность

### Транзакции
- Добавление транзакции: сумма, категория, тип (доход/расход), дата, описание (опционально)
- Удаление транзакции
- Список транзакций с фильтрацией по месяцу

### Категории
- Предустановленные категории расходов: Еда, Транспорт, Жильё, Развлечения, Здоровье, Одежда, Прочее
- Предустановленные категории доходов: Зарплата, Фриланс, Прочее

### Дашборд
- Текущий баланс (доходы - расходы)
- Круговая диаграмма расходов по категориям (за выбранный месяц)
- Столбчатая диаграмма доходы vs расходы по месяцам (за последние 6 месяцев)

### Хранение
- Все данные в localStorage
- Автосохранение при каждом изменении

## Структура страниц
1. **Дашборд** — баланс + графики
2. **Транзакции** — список + форма добавления

Навигация через табы в верхней части приложения.

## Модель данных

```ts
type TransactionType = "income" | "expense";

interface Transaction {
  id: string;
  amount: number;
  type: TransactionType;
  category: string;
  date: string;       // ISO date string (YYYY-MM-DD)
  description?: string;
}
```

## UI
- Минималистичный, светлая тема
- Адаптив: мобильные + десктоп
- Цвета: зелёный для доходов, красный для расходов
